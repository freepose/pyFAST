Dataset Layer
dataset/
Handles raw data preparation and provides unified loaders for various time series formats.
  • Management and preparation scripts for datasets.
  • Loaders for dense & sparse series.
  • Supports SMC, SMI, SMT, SST, and MMT multisource dataset families.
`manage_*.py` `prepare_*.py`
Preprocessing
fast/data/
Focuses on turning raw data into model-ready tensors.
  • Time-series windowing, padding, and truncation.
  • Mask generation for missing and irregular data points.
  • Sparsity-aware scaling and normalization.
`mask.py` `scale.py` `processing/`

Typical Research Workflow
1. Prepare data with scripts in dataset/.
2. Define an experiment in experiment/config_json/.
3. Launch a run using an entrypoint from example/.
4. Analyze results from metric/ and fast/visualize.py.
Model Zoo
model/
A comprehensive collection of models for forecasting and imputation.
  • Base blocks: Attention, MLP, RNNs in `model/base/`.
  • Forecasting: DLinear, TCN, Transformers, GNNs in `model/mts/`.
  • Imputation: GRU-D, BRITS, RITS in `fast/impute/`.

Evaluation
metric/
Provides tools for robust model performance assessment.
  • Unified metric definitions (MSE, MAE, etc.).
  • Mask-aware evaluation for sparse/missing targets.
  • Experiment-level aggregation and reporting.
